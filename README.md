# MCA × Cobblemon — Issue Tracker

This repository is the public issue tracker for the **MCA × Cobblemon** mod.
The source code is maintained separately in a private repository.

---

## Before opening an issue

Please check the following first:

- Search [existing issues](../../issues) to see if your bug has already been reported
- Make sure you are using the correct versions listed in the compatibility table below
- Make sure all required mods are installed — missing dependencies are the most common cause of crashes

### Required mods

| Mod | Where to download |
|---|---|
| Minecraft Comes Alive Reborn | [CurseForge](https://www.curseforge.com/minecraft/mc-mods/minecraft-comes-alive-reborn) |
| Cobblemon | [CurseForge](https://www.curseforge.com/minecraft/mc-mods/cobblemon) |
| Kotlin for Forge | [CurseForge](https://www.curseforge.com/minecraft/mc-mods/kotlin-for-forge) |

### Compatibility

| Minecraft | NeoForge | MCA | Cobblemon |
|---|---|---|---|
| 1.21.1 | 21.1.x | 7.x | 1.7.x |

---

## Reporting a bug

Open a [new issue](../../issues/new) and include the following:

1. **What happened** — describe the bug clearly
2. **What you expected to happen**
3. **Steps to reproduce** — the more specific the better
4. **Your mod list** — a full list of every mod you have installed
5. **Your log file** — attach the full `latest.log` from your `.minecraft/logs/` folder. Reports without a log file are much harder to investigate and may be closed

---

## Feature requests

Feature requests are welcome. Open a [new issue](../../issues/new) and prefix the title with `[Feature]`. Keep in mind this mod is in beta and core stability is the current priority.

---

## This is a beta mod

MCA × Cobblemon is in active development. Crashes and incomplete features are possible. **Always keep a backup copy of your world.** Save data may not be forward-compatible between versions.

---

## This repository is for issues only

Please do not open pull requests here — the source code is not hosted in this repository. For general questions or discussion, use the Comments section on the [CurseForge page](https://www.curseforge.com/minecraft/mc-mods/mca-cobblemon).
